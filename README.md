# cli
Outpost CLI


# Setup Dev
```
pip install --editable .

outpostcli login [-t api_token]
```

# Get Current User
```
outpostcli user [-t api_token]
```

# List Inferences
```
outpostcli inferences list [-e entity] [-t api_token]
```

# Get Inference
```
outpostcli inference get <inf_name> [-e entity] [-t api_token]
```
